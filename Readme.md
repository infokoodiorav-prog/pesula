# [KülaPesula]  

KülaPesula is a fictional project created for portfolio/demo purposes.

This is a multi-page front-end project demonstrating a multilingual e-commerce and booking system. It is built as a demo to showcase interactive elements, responsive design, and user interface functionality.

## Project Overview

- **Multi-page layout**: home, products, booking, contact, etc.  
- **Multi-language support**: allows switching between multiple languages (translations implemented for demonstration purposes).  
- **E-commerce demo**: Users can add items to the cart and proceed to “checkout”; payment is simulated and no real transactions occur.  
- **Booking system**: Users can make a booking; system confirms the booking, but no real availability tracking is implemented.  
- **Responsive design**: Works on both desktop and mobile screens.  

## Interactive Features

- Clickable buttons and popups for product details  
- Dynamic form handling for booking and checkout  
- Navigation menu with burger menu for mobile devices  
- Multi-language content switching  

## Technologies Used

- HTML, CSS, JavaScript  
- JSON files for translations  
- No backend; all functionality is simulated in the front-end
